### :memo: A aplicação permite:

*   Cadastro de clintes;
*   Cadastro de transações;
*   Geração de extrato;
*   Listagem de dados cadastrados;
*   Inputs de busca (Clientes e transações);
*   Atualização de informações cadastradas;
*   Exclusão lógica de informações cadastradas.
